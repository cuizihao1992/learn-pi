One Sentence Model

先记住这一句

Pi Agent 运行一次任务时，模型只负责“根据上下文决定下一步”，比如要读文件、改文件、运行测试； 真正的文件系统操作、命令执行、状态管理、会话保存、上下文压缩和前端更新，都由 Pi 的运行时负责。 

用户提示词
  -> AgentSession 组装上下文和工具
  -> AgentLoop 调用模型
  -> 模型返回 toolCall
  -> Pi 执行工具并生成 toolResult
  -> toolResult 再发给模型
  -> 最终回答通过事件返回前端
