读源码练习任务

按顺序完成这些任务，你会同时理解 Pi Agent 的“请求进入、模型响应、工具执行、会话持久化、上下文压缩”五条主线。

跟踪一次 prompt 从 `main.ts` 进入，找到 `AgentSession.prompt()`，再追到 `Agent.run()` 和 `runLoop()`。
新增一个只读工具 模仿 `grep.ts` 或 `ls.ts`，注册到 `tools/index.ts`，观察工具定义和工具实例的分离。
理解压缩触发 阅读 `compaction.ts`，比较手动压缩、阈值压缩和 overflow recovery。
导出会话 HTML 从 `export-html/index.ts` 看消息、工具结果和 ANSI 渲染如何转换成分享页面。
