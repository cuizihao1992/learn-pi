Globalization

全球化以后，钱和规则怎么流动

在一个国家内部，货币、财政、税收、工资和价格主要受本国制度影响。全球化以后，还要加入汇率、贸易、资本流动、跨国公司、美元体系、国际组织和地缘政治。

一、全球化改变了什么

国内经济
  + 贸易：商品和服务跨国流动
  + 汇率：不同货币之间换算
  + 资本：钱跨国寻找收益和安全
  + 技术：标准、专利、平台跨国扩散
  + 供应链：生产拆到多个国家
  + 规则：关税、制裁、结算、监管、协议
变量影响例子
贸易改变商品价格、产业结构和就业进口便宜商品降低消费成本，出口产业创造岗位。
汇率改变出口竞争力和进口成本本币贬值利好出口，但进口原料更贵。
资本流动改变资产价格、融资成本和汇率外资流入推高股债汇，流出带来压力。
供应链降低成本，也增加外部依赖芯片、能源、航运扰动会影响全球价格。
技术标准决定谁掌握高利润环节操作系统、芯片架构、通信标准、支付网络。

二、全球规则是谁定的

全球没有一个统一政府。规则由实力、协议、市场和历史共同形成。

  • 大国实力：市场规模、军事、科技、金融能力决定谈判权。
  • 国际组织：WTO 管贸易规则，IMF 关注国际金融稳定，世界银行做发展融资，BIS 连接央行和金融监管。
  • 美元体系：美元作为主要储备和结算货币，使美国利率和金融条件影响全球。
  • 跨国公司：通过供应链、采购标准、专利和平台规则影响全球分工。
  • 金融市场：资本会用价格表达对国家政策、风险和收益的判断。
  • 双边和多边协议：贸易协定、投资协定、关税安排塑造边界。

三、为什么美元体系重要

很多国际贸易、债务、储备和金融资产以美元计价。于是美元利率、美元流动性和美国金融市场会外溢到全球。

美国加息
  -> 美元资产收益率上升
  -> 全球资金可能回流美元
  -> 新兴市场汇率承压
  -> 外币债务成本上升
  -> 本国央行面临稳汇率和稳增长的取舍

四、全球化对普通人的影响

影响路径可能好处可能压力
消费品进口商品更便宜、选择更多本国产业竞争加剧
出口产业创造就业和收入依赖海外需求和贸易环境
资本流入融资更容易、资产上涨泡沫和突然流出风险
技术扩散生产率提高低技能岗位被替代
产业链分工国家能参与全球生产利润可能集中在高端环节

五、理解全球新闻的模板

看到一条全球新闻，问：
1. 它影响贸易还是资本？
2. 它影响汇率还是利率？
3. 它影响供应链哪个环节？
4. 哪些国家、行业、企业受益？
5. 哪些劳动者和消费者承压？
6. 这是短期冲击，还是长期规则改变？
