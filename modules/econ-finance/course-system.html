Course System

把经济金融学成一门系统课

这个模块不按“概念词典”堆知识点，而按系统课程组织：先看社会部门和钱的循环，再学价格机制、财政货币、城市空间、全球规则，最后落到金融资产、周期判断和投资约束。

一、课程主线

社会部门
  -> 金钱流动
  -> 价格与激励
  -> 财政、货币、信用
  -> 城市与全球约束
  -> 现金流、利率、风险
  -> 宏观周期与资产价格

二、七个课程单元

单元核心问题对应页面最小输出
1. 社会系统家庭、企业、政府、金融、城市、海外部门如何互相约束？社会运行总图画出六部门循环图。
2. 收入与分配工资、利润、租金、利息、税收和转移支付如何形成？金钱流动与分配解释一个人的收入为什么受行业、城市和资产影响。
3. 价格机制供需、弹性、成本、竞争和制度如何决定价格？经济学基础用供需图解释一个现实价格变化。
4. 政策与信用财政、央行、商业银行和信用周期如何改变需求与资产价格？财政与货币画出降息或财政扩张的传导链。
5. 空间与全球城市集聚、房租、贸易、汇率和美元体系如何改变机会？城市经济 / 全球化规则比较一个大城市和小城市的收入成本结构。
6. 金融与周期现金流、利率、风险和宏观周期如何进入资产定价？金融学基础 / 宏观周期 / 利率折现深挖解释利率上升为什么会压制高久期资产。
7. 案例与术语如何把术语、指标和现实案例连成可复用分析框架？案例分析 / 术语地图用同一套模板拆一条财经新闻。

三、每章固定学习法

  1. 先写一句话模型：这个章节到底解释什么。
  2. 再画因果链：变量 A 通过什么机制影响变量 B。
  3. 再找现实案例：新闻、城市、行业、资产价格任选一个。
  4. 最后写反例：这个解释在什么条件下会失效。

四、系统课的考核标准

层级标准自测问题
看懂能读懂页面里的术语和图。能不能用自己的话解释“信用扩张”？
画出能把变量关系画成链路。能不能画出工资、消费、企业收入和就业的循环？
迁移能解释一个新案例。能不能用同一套框架分析房租、油价或汇率？
质疑能说出模型边界。能不能指出“降息一定利好股市”为什么太粗糙？

五、和交易学习的边界

经济金融系统课不是荐股课。它提供的是解释框架：现金流、折现率、风险偏好、流动性、信用和政策预期如何影响价格。真正交易还要回到仓位、期限、止损、交易成本、样本外验证和个人约束。
