经济金融术语不是散点，而是一张传导图

很多财经新闻难懂，是因为术语彼此连接：GDP 影响收入和盈利，通胀影响利率，利率影响折现率和信用，信用影响需求与资产价格。先背词不如先看连接关系。

一、总图

增长 GDP / 收入 / 就业
  -> 企业盈利和居民消费
  -> 通胀 CPI / PPI
  -> 利率 / 流动性 / 信用
  -> 折现率 / 风险溢价 / 杠杆
  -> 债券、股票、房产、汇率
  -> 财富效应和融资条件
  -> 再影响增长

二、宏观指标组

术语一句话常连接到
GDP一段时间内经济生产的总价值。收入、企业盈利、就业、财政收入。
CPI居民消费价格变化。实际购买力、工资谈判、央行政策。
PPI工业生产者价格变化。企业成本、利润率、CPI 传导。
失业率劳动市场松紧的总量指标。工资、消费、政策托底、社会压力。
PMI企业景气度调查指标。订单、库存、生产、经济拐点。

三、金融定价组

术语一句话常连接到
现金流资产未来能分给持有人的钱。股票估值、债券本息、公司价值。
折现率把未来现金流折回今天的利率。利率、风险溢价、估值。
久期资产价格对利率变化的敏感度。债券、成长股、高估值资产。
风险溢价承担不确定性要求的额外收益。股票、信用债、流动性压力。
流动性资产变成现金的速度和成本。市场波动、折价抛售、央行工具。

四、信用政策组

术语一句话常连接到
基础货币央行创造的高能货币。银行准备金、支付清算、流动性。
信用扩张贷款和融资更容易增加。投资、房产、企业现金流、资产价格。
财政赤字政府支出超过收入的部分。国债、总需求、债务可持续性。
杠杆用借来的钱放大资产暴露。收益、亏损、保证金、金融风险。
汇率两种货币之间的相对价格。贸易、资本流动、外债、进口通胀。

五、看到术语时的三问

  1. 它首先影响现金流，还是影响折现率？
  2. 它是改变实体经济的收入成本，还是改变金融市场的风险偏好和流动性？
  3. 它是已经发生的数据，还是市场正在交易的预期？
术语变成机制看利率、折现率、久期如何连成估值链路
