Deep Dive

利率如何通过折现影响资产价格

这是经济金融模块先打穿的关键点：资产价格不是只由“好公司、坏公司”决定，而是由未来现金流和折现率共同决定。利率上升时，未来的钱折回今天会变少，所以高久期资产通常更受压制。

一句话模型

资产价格
  = 未来现金流折回今天的总和

利率上升
  -> 折现率上升
  -> 同样一笔未来现金流的现值下降
  -> 久期越长、现金流越靠后的资产越敏感

需要补的前置知识

概念一句话为什么重要
现金流资产未来能带来的钱。估值的分子。
折现率把未来现金流折回今天使用的收益率要求。估值的分母。
无风险利率接近没有违约风险的基础收益率。很多资产定价以它为锚。
风险溢价承担不确定性要求的额外补偿。公司风险、行业风险、流动性风险都会抬高它。
久期价格对利率变化的敏感度。现金流越远，久期通常越长。

底层链路

央行加息 / 市场利率上行
  -> 无风险利率上升
  -> 投资者要求的最低回报上升
  -> 折现率 = 无风险利率 + 风险溢价
  -> 未来现金流现值下降
  -> 估值倍数下降
  -> 股票、债券、房产、成长型资产重新定价

一个最小例子

假设某资产一年后确定收到 110 元。今天愿意出多少钱，取决于你要求的年收益率。

要求收益率计算今天价值
5%110 / 1.05104.76
10%110 / 1.10100.00
20%110 / 1.2091.67

未来现金流没变，只是折现率变了，今天的价格就变了。这就是“加息压估值”的最小数学直觉。

为什么成长股更敏感

成熟公司可能现在就有利润和现金流；成长公司很多价值来自更遥远的未来。越远的现金流，被折现率影响越大。

资产 A：今年、明年、后年都有稳定现金流
资产 B：前几年现金流少，主要现金流在 8 到 10 年后

当折现率上升：
  -> 远期现金流被打折更多
  -> 资产 B 的估值下降更明显
  -> 这就是高久期资产对利率更敏感

和债券、股票、房产的关系

资产利率上升时的典型影响背后机制
债券旧债价格下跌。新债收益率更高，旧债票息吸引力下降。
成长股估值倍数容易被压缩。价值集中在远期现金流，久期较长。
银行不一定单向受益或受损。息差可能改善，但信用风险和坏账也可能上升。
房产购买力和估值承压。按揭成本上升，租金现值下降，杠杆需求下降。

常见误解

  • 误解一：加息一定让所有股票跌。不一定。若盈利预期同步大幅改善，现金流上调可能抵消折现率上升。
  • 误解二：利率只影响债券。不对。只要资产价格来自未来现金流，折现率就会影响它。
  • 误解三：估值就是市盈率。市盈率只是结果指标，背后仍是增长、利润率、折现率和风险溢价。
  • 误解四：利率越低资产越安全。低利率会推高估值，也可能鼓励杠杆和泡沫。

失败模式：为什么现实里不是公式这么简单

复杂因素会怎样扰动结论
现金流预期变化加息可能压需求，也可能发生在经济强劲阶段。
风险溢价变化恐慌时风险溢价上升，资产价格可能比利率变化暗示的跌得更多。
流动性冲击投资者被迫卖出时，好资产也可能被折价抛售。
政策预期市场交易的是未来利率路径，不只是今天的利率。

自测问题

  1. 为什么同样是加息，短久期债券通常比长期债券更抗跌？
  2. 为什么“盈利增长很好”的公司也可能在利率上行阶段估值下跌？
  3. 如果一家公司未来现金流主要发生在十年后，它和一年后收钱的资产谁更怕加息？
  4. 为什么只看市盈率，容易忽略利率和风险溢价的变化？
  5. 如果央行降息但经济衰退严重，资产价格一定会上涨吗？为什么？

回到模块复盘

理解利率折现后，经济金融模块里的很多概念会连起来：通胀影响利率，利率影响折现率，折现率影响资产价格，资产价格又通过财富效应、融资条件和风险偏好影响实体经济。
