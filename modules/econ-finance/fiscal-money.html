Fiscal and Monetary System

政府的钱、银行的钱、老百姓的钱怎么连起来

“钱是不是来自税收？”这个问题要分清财政和货币。税收是政府收入和再分配的重要来源，但现代货币还来自央行基础货币和商业银行信用创造。社会里的钱，是财政系统和银行系统共同运转的结果。

一、财政循环

政府收税 / 发债 / 国有资产收入
  -> 形成财政资金
  -> 支付公务员工资、社保、医保、教育、基建、采购、补贴
  -> 居民和企业获得收入
  -> 消费和投资扩大企业收入
  -> 企业和居民再缴税
  -> 回到财政系统
财政工具作用分配影响
个人所得税从个人收入中征税影响收入再分配和劳动激励。
增值税对商品服务增值部分征税影响商品价格和企业现金流。
社保缴费养老、医疗、失业等筹资连接当期劳动收入和未来保障。
政府采购政府购买商品服务直接形成企业订单和就业。
转移支付社保、补贴、救助、地方转移把购买力转给特定人群或地区。
政府债务把未来财政收入提前使用支持当期支出，也形成未来偿债约束。

二、银行信用循环

很多人以为银行只是“把存款借出去”。更准确地说，商业银行发放贷款时，会同时创造一笔贷款资产和一笔存款负债。贷款让借款人账户上出现可支付的钱，还款则让这部分存款消失。

银行发放贷款
  -> 借款人账户出现存款
  -> 借款人买房、经营、投资或消费
  -> 钱流向卖方、工人、供应商
  -> 形成更多存款和收入
  -> 借款人未来用收入还本付息
  -> 信用收缩或循环继续

这就是为什么房地产、企业投资、地方建设、消费金融都和信用周期密切相关。信用扩张时，钱更容易流动，资产价格和投资更容易上行；信用收缩时，现金流压力会放大。

三、央行在做什么

  • 发行和管理基础货币：保证银行系统有结算基础。
  • 影响利率：通过政策利率和流动性影响融资成本。
  • 维护金融稳定：在流动性紧张时提供支持。
  • 管理预期：通过政策沟通影响市场对未来利率和通胀的判断。
  • 外汇和国际收支：在一些制度下，外汇流入流出也会影响基础货币。

四、财政政策和货币政策的区别

政策主要部门直接工具主要影响
财政政策政府财政部门税收、支出、赤字、债务总需求、公共服务、收入再分配、产业方向。
货币政策央行利率、准备金、公开市场操作、流动性工具融资成本、信用扩张、通胀预期、资产估值。

财政更像“政府直接花钱和收钱”；货币更像“改变钱和信用的价格与数量”。两者会互相影响：政府发债需要金融市场承接，央行利率会影响政府债务成本，财政支出也会影响通胀和货币政策。

五、钱流到普通人手中的主要路径

  1. 企业销售商品服务，获得收入，支付工资。
  2. 政府支出形成工资、补贴、社保、采购款。
  3. 银行贷款支持企业和个人支出，转化为别人的收入。
  4. 资产收入通过租金、利息、分红、资本利得进入家庭。
  5. 对外贸易和投资带来的收入进入国内企业和劳动者。
