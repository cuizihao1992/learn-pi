Social System

社会到底是怎么运行的：一张系统总图

如果你想对社会运行有更深刻的认识，不能只学“经济学”或“金融学”某一个点。要把社会看成一个多层系统：家庭提供劳动和消费，企业组织生产，政府制定规则和再分配，金融体系跨时间调配资金，城市承载人口和产业，国际体系改变边界。

二、社会运行的主循环

家庭 -> 提供劳动和消费需求
企业 -> 组织生产并支付工资、租金、利息、利润
政府 -> 提供规则、公共品和再分配
金融 -> 把储蓄、信用和风险跨时间配置
城市 -> 把人、地、产业、公共服务集中到空间里
全球 -> 通过贸易、汇率、资本、技术改变国内价格和机会

四、学习社会运行的正确顺序

  1. 先学经济学：价格、供需、激励、市场失灵。
  2. 再学金融学：现金流、利率、风险、信用。
  3. 再学财政货币：政府钱怎么来，银行如何创造信用。
  4. 再学城市经济：为什么机会、工资、房租和公共资源集中。
  5. 再学国际经济：贸易、汇率、美元体系、资本流动和全球规则。
  6. 最后回到历史和制度：产权、土地、税制、教育、户籍、产业政策如何长期塑造分配。

五、一个成熟的社会观察问题模板

看到一个社会现象，按这 8 个问题拆：
1. 谁是参与者？
2. 他们分别有什么资源和约束？
3. 钱从哪里来，流向哪里？
4. 价格由什么供需关系决定？
5. 政府规则改变了什么激励？
6. 金融信用放大了什么？
7. 空间和城市位置有什么影响？
8. 全球贸易、汇率、资本和技术有没有参与？
