Macro Cycle

宏观指标与经济周期怎么读

宏观经济不是一堆新闻词汇。你要把它看成一个相互连接的系统：增长、通胀、就业、利率、信用、财政、汇率和资产价格不断互相影响。

一、宏观系统图

居民收入和消费
  -> 企业收入和投资
  -> 就业和工资
  -> 通胀压力
  -> 央行利率和流动性
  -> 信用扩张或收缩
  -> 资产价格
  -> 财富效应和融资条件
  -> 再影响消费与投资

二、关键指标

指标看什么常见解释
GDP经济总产出增长加速通常利好收入和盈利，但也可能带来通胀和加息压力。
CPI / PPI消费价格 / 工业品价格通胀高会影响实际购买力和央行政策。
失业率劳动力市场松紧低失业可能推高工资和通胀，高失业说明需求不足或结构问题。
利率资金价格利率上升提高折现率，压制高估值资产。
信用贷款和融资扩张信用扩张推动投资和资产价格，信用收缩带来压力。
汇率货币相对价格影响进出口、资本流动和外币债务。

三、周期的四种典型状态

状态增长通胀政策倾向资产含义
复苏回升较低宽松逐渐退出风险资产通常改善。
过热高高收紧盈利好但估值受压。
滞胀弱高两难股票债券都可能难受。
衰退弱回落宽松债券可能受益，股票看盈利下修程度。

四、政策怎么影响市场

  • 降息：降低融资成本和折现率，但也可能说明经济走弱。
  • 加息：压制需求和估值，但也可能说明经济过热。
  • 财政扩张：提高总需求，但可能增加赤字和债务压力。
  • 信用宽松：推动融资和资产价格，但过度会积累金融风险。

五、读宏观新闻的方法

看到一个新闻，不要只问“利好还是利空”。要问五个问题：它影响增长还是通胀？影响利率还是信用？影响现金流还是折现率？影响哪个行业？市场之前是否已经预期？这样你会从情绪判断变成机制判断。
