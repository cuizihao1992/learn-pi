三、指标怎么算

SMA：简单平均

SMA5 就是最近 5 个收盘价的普通平均。它能过滤噪声，但一定滞后。

SMA5 = (C1 + C2 + C3 + C4 + C5) / 5

EMA：给新价格更高权重

EMA5 比 SMA5 更敏感，因为最新价格权重更高。

alpha = 2 / (N + 1)
EMA = price * alpha + previousEMA * (1 - alpha)

MACD：短均线减长均线

这里用教学参数 5 / 10 / 4。DIF 看短期 EMA 是否强于长期 EMA，DEA 是 DIF 的平滑。

DIF = EMA5 - EMA10
DEA = EMA(DIF, 4)
Histogram = DIF - DEA

RSI：上涨幅度和下跌幅度比较

这里用教学参数 6。RSI 高说明最近上涨幅度相对更强，低说明下跌幅度相对更强。

RS = avgGain / avgLoss
RSI = 100 - 100 / (1 + RS)

五、怎么看失败模式

  • 震荡行情：均线和 MACD 会频繁交叉，很多信号只是噪声。
  • 强趋势：RSI 可以长期高位或低位，不能机械理解成马上反转。
  • 样本太短：长周期指标还没稳定，前几天的值参考意义有限。
  • 只看指标：指标不包含财报、政策、流动性、仓位和风险收益比。
