看见一次语义检索怎么发生

这个实验用一个很小的“教学向量模型”模拟 embedding。它不会调用真实模型，但会展示 RAG 第一步的核心动作：把 query 和文档片段映射到同一组语义维度，再用余弦相似度排序。

这个 demo 在模拟什么

真实 RAG：
query -> embedding model -> query vector
documents -> chunk -> embedding model -> chunk vectors
query vector vs chunk vectors -> topK evidence
topK evidence + user question -> LLM answer

这个教学 demo：
query -> 关键词映射到 5 个语义维度
candidate texts -> 同样映射到 5 个语义维度
余弦相似度 -> 排序展示

注意边界

  • 真实 embedding 是高维向量，不是这里的 5 个手写维度。
  • 真实语义相似度来自模型训练，不是关键词规则。
  • 真实 RAG 还需要 chunk、metadata、BM25、rerank、权限、引用和评估。
  • 这个 demo 的价值是帮助你先看见“检索增强生成”的第一步。
