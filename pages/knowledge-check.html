16 个问题检验你的理解

1. 模型为什么不能直接改文件？

因为模型只输出文本或结构化 toolCall。真实文件操作必须由程序工具执行。

2. toolResult 为什么要放回 messages？

模型需要基于工具结果决定下一步，形成多轮推理。

3. Agent 和 AgentSession 最大区别是什么？

Agent 是通用运行时状态；AgentSession 是 coding-agent 产品运行时中心。

4. 自研模型接入 Agent 最省事路线是什么？

实现 OpenAI-compatible `/v1/chat/completions` adapter。

5. JSON 是模型天然输入吗？

不是。JSON 是 API 协议，服务层会把它转成模型 prompt/token。

6. Cesium Entity 和 Primitive 有什么区别？

Entity 是业务抽象，Primitive 更接近渲染层，会产生命令。

7. DrawCommand 包含哪些核心信息？

vertexArray、shaderProgram、uniformMap、renderState、pass、boundingVolume。

8. Cesium 为什么需要 commandList？

Scene 可以统一裁剪、排序、分 pass、执行命令。

9. OpenLayers 的 Layer 和 Source 为什么分开？

Source 负责数据加载，Layer 负责视觉表达。这样同一种数据源可以用不同方式展示，Layer 也能统一参与渲染调度。

10. OpenLayers View 主要决定什么？

决定中心点、分辨率/zoom、旋转和投影，进而影响需要加载哪些瓦片、哪些要素处在视口内。

11. 3D Gaussian Splatting 的场景表示是什么？

是一组可优化的 3D 高斯椭球，每个高斯通常包含位置、尺度/旋转或协方差、不透明度、颜色或球谐系数。

12. 高斯泼溅训练时为什么需要可微 rasterizer？

因为渲染图要和真实照片计算 loss，再把梯度传回每个高斯参数，决定它该移动、变形、变透明还是改变颜色。

13. K 线本质上记录什么？

记录一个周期内的开盘、收盘、最高、最低，是市场成交结果的压缩表达，不是未来预测机器。

14. 技术指标为什么会失效？

因为指标大多来自历史价格和成交量，存在滞后、噪声、市场环境变化、低流动性和重大消息冲击等问题。

15. 一个买入信号至少要拆成哪些部分？

环境、位置、触发、确认、失效、仓位和复盘。没有失效条件的信号不是计划，只是愿望。

16. Pi、Cesium、OpenLayers、高斯泼溅、股票 K 线的共同架构思想是什么？

高层意图或现象先转成结构化中间对象，再由底层规则统一解释、调度或执行。可扩展的关键是边界清晰，各层各司其职。
