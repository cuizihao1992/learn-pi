每学完一章要能回答什么

  • Pi：模型到底做决策还是执行？工具结果为什么要回填？
  • Mini Agent：最小 Agent Loop 为什么必须有 toolResult 回填？
  • LLM 面试：记忆为什么要分层？RAG 为什么不只是向量库？长上下文为什么不能完全替代 RAG？
  • Cesium：Entity 为什么不能直接 draw？DrawCommand 包含哪些 WebGL 信息？CustomShader、Material、PostProcessStage 分别插入哪一层？
  • OpenLayers：Layer 和 Source 为什么分开？View 如何决定瓦片和要素渲染？
  • Gaussian Splatting：为什么一个 3D 场景可以表示成很多高斯椭球？训练和渲染阶段分别优化什么？
  • 经济金融：为什么价格能协调供需？为什么利率会影响资产估值？为什么风险必须有补偿？
  • 股票 K 线：指标到底在加工什么历史数据？一个买入信号为什么必须包含环境、位置、触发、确认、失效和仓位？
  • 共同点：为什么大型系统都喜欢中间命令层、索引层、调度层和可观测性？
