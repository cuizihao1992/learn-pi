四个学习模块怎么走

现在有四个并列模块：Pi Agent 负责读懂开源 Agent 架构，Mini Agent 负责从零实现自己的 Agent，Cesium 负责三维渲染链路，OpenLayers 负责二维地图渲染链路。四者共同点都是高层意图先转成中间结构，再由底层统一调度执行。

每学完一章要能回答什么

  • Pi：模型到底做决策还是执行？工具结果为什么要回填？
  • Mini Agent：最小 Agent Loop 为什么必须有 toolResult 回填？
  • Cesium：Entity 为什么不能直接 draw？DrawCommand 包含哪些 WebGL 信息？
  • OpenLayers：Layer 和 Source 为什么分开？View 如何决定瓦片和要素渲染？
  • 共同点：为什么大型系统都喜欢中间命令层？
