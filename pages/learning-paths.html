Learning Paths

不要从页面开始，从目标开始

这个网站已经有 11 个模块。小白最容易卡住的地方不是资料不够，而是不知道先学哪条线。这里把内容按目标重组：想开发 Agent、准备面试、学地图渲染、做数字孪生、入门交易系统或理解科学技术，可以直接选路线。

路线一：Agent 产品实战

适合目标：想从小白走到“能开发自己的 Agent”。

  1. Pi Agent 总览：先看真实 Agent 工程有哪些层。
  2. Pi 完整运行流程：理解 prompt、model、toolCall、toolResult、final answer 的闭环。
  3. Mini Agent MVP：用 Node 写出最小 Agent Loop。
  4. 最小 Agent Loop 深挖：把模型决策、工具执行、结果回填和终止条件拆成一个闭环。
  5. Agent Evaluation Demo：用 trace 判断工具选择、参数、toolResult grounding 和最终回答。
  6. Agent Safety Demo：理解路径边界、命令白名单、写文件确认和密钥外传拦截。
  7. 模型接口兼容：理解 OpenAI-compatible、自研模型中间层、Anthropic 差异。
  8. 从零实战路线：把命令行 Agent 做成带前端、记忆和 RAG 的小产品。
  9. AI 记忆机制：给 Agent 加短期记忆、用户画像和长期情景记忆。
  10. RAG 数据工程：给 Agent 接知识库。
验收标准：
用户在网页输入任务
  -> Agent 能选择工具
  -> 工具结果回填给模型
  -> 前端能看到工具调用过程
  -> Agent 能记住项目偏好
  -> Agent 能从本地文档检索答案

路线二：LLM / Agent 系统课

适合目标：面试算法岗、Agent 应用岗、RAG 工程岗、大模型系统岗。不要先背名词，先用一个具体问题打穿“语义表示 -> 证据召回 -> prompt 生成 -> 评估”的闭环。

  1. 课程入口：先建立 LLM / Agent 系统链路。
  2. Embedding -> RAG 深挖：用具体例子理解一句话如何找到正确证据。
  3. Mini RAG Demo：观察文档切片、topK 召回、prompt 拼接和引用回答。
  4. RAG Evaluation Demo：用 Recall@K、MRR 和忠实度判断系统错在哪里。
  5. RAG 向量数据工程：数据清洗、语义切片、向量库、混合检索、rerank、评估。
  6. 长期陪伴 AI 记忆机制：分层存储、异步更新、冲突和遗忘。
  7. 长上下文 vs RAG：讲清二者互补关系和 Long RAG。
  8. Transformer 长文本优化：O(n^2)、KV Cache、FlashAttention、GQA、PagedAttention、RoPE/ALiBi。
  9. 答题模板：把知识组织成两分钟答案。
答题骨架：
先给结论
  -> 画系统链路
  -> 讲工程取舍
  -> 讲失败模式
  -> 讲评估指标
  -> 落到真实系统怎么做

路线三：GIS / 渲染

适合目标：想理解地图、WebGL、三维引擎、渲染命令如何到 GPU。

  1. OpenLayers 总览：先学传统二维 GIS 的 Map、View、Layer、Source。
  2. OpenLayers 渲染路径：看瓦片、矢量、Canvas/WebGL 如何画出来。
  3. Mapbox 总览：理解 source、layer、style、vector tiles。
  4. Mapbox 与 Cesium 对比：理解为什么 Mapbox 在二维场景通常更轻快。
  5. Cesium 总览：进入三维地球和大规模 3D Tiles。
  6. Cesium 完整渲染路径：从 Viewer、Scene、Primitive 到 DrawCommand。
  7. Cesium 自定义着色器：理解 shader 最终如何进入渲染命令。

路线四：数字孪生城市

适合目标：想做城市级地图可视化、应急系统、园区管理、设备监控。

  1. 数字孪生城市总览：看完整系统分层。
  2. 地图底座：理解 Mapbox、Cesium、OpenLayers 如何分工。
  3. 三维资产与数据：理解 3D Tiles、BIM、倾斜摄影、点云和高斯泼溅。
  4. 实时数据与 Agent：把告警、传感器、工具调用串起来。
  5. 前后端架构：理解 API、队列、数据库、前端状态。
  6. MVP 路线：先做最小闭环，不一开始做大屏。

路线五：经济金融与社会系统

适合目标：想理解钱如何流动、工资房租如何形成、政府和市场如何共同分配，再入门金融市场和量化交易系统。

  1. 经济学与金融学总览：先建立价格、资源、现金流、利率和风险框架。
  2. 社会运行总图：把家庭、企业、政府、金融、城市和全球部门放进一张系统图。
  3. 金钱流动与分配：理解工资、房租、物价、税收、政府支出、银行信贷和全球规则。
  4. 财政与货币循环：理解税收、国债、央行、银行贷款和信用创造。
  5. 城市经济：理解一线城市和小地方的工资、房租、物价和机会差异。
  6. 全球化规则：理解贸易、汇率、美元体系、资本流动和国际组织。
  7. 经济学基础框架：理解稀缺、机会成本、供需、弹性、市场失灵。
  8. 金融学基础框架：理解现金流、时间价值、折现率、风险收益。
  9. 利率折现深挖：用最小数学例子理解加息为什么压估值。
  10. 宏观指标与周期：理解 GDP、通胀、就业、利率、信用和资产价格。
  11. 股票 K 线总览：再建立价格、成交量、趋势、风控框架。
  12. K 线基础：理解单根 K 线和组合形态。
  13. 指标底层逻辑：理解均线、MACD、RSI、KDJ、布林带到底在加工什么。
  14. K 线指标 Demo：输入价格序列，观察 SMA、EMA、MACD、RSI 的计算过程。
  15. 买卖信号整理：按长短线组织信号，而不是孤立看指标。
  16. 风控日志：学会仓位、止损、复盘。
  17. 量化交易总览：从人工判断走到策略系统。
  18. 量化系统架构：理解数据、研究、回测、实盘、风控、监控。

路线六：科学技术

适合目标：想保持好奇心，同时理解高斯、三维重建、空间电源这些硬核主题。

  1. Gaussian Splatting 总览：理解照片到 3D 场景。
  2. 训练与渲染路径：理解相机位姿、稀疏点云、高斯参数、可微渲染。
  3. 调试手册：知道训练失败时看什么。
  4. 核电池总览：区分 RTG、贝塔伏特电池和普通电池。
  5. 工作原理：理解放射性衰变、热电转换和能量密度。
  6. 卫星与原子钟：分清原子钟是计时设备，不是核燃料电池。
