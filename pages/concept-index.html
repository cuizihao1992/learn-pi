Agent 与大模型

概念一句话入口
toolCall模型提出“我要调用哪个工具和参数”。工具调用
toolResult运行时执行工具后，把真实结果回填给模型。Mini Agent
OpenAI-compatible让不同模型伪装成同一种 API 协议。模型接口
Embedding把文本、图片、代码等对象变成可比较的语义向量。Embedding 与多模态
RAG先检索证据，再让模型回答。RAG 工程
多模态模型把图像、音频、视频等模态接入语言模型的理解和生成能力。Embedding 与多模态
KV Cache推理时保存历史 token 的 Key/Value，避免重复计算。长文本优化
