Deep Dive Roadmap

每个模块只先打穿一个关键点

这个计划不是把模块做大而全，而是为每个模块选一个最能代表底层原理的点，用一句话模型、底层链路、具体例子、最小实验、常见误解和自测题讲透。

深挖模板

一个关键点
  -> 一句话模型
  -> 需要补的前置知识
  -> 底层链路图
  -> 具体例子
  -> 最小实验 / 可观察 demo
  -> 常见误解
  -> 自测问题
  -> 回到模块复盘
