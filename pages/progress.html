Agent 主线打卡

  • 能解释 prompt -> model -> toolCall -> toolResult -> final answer。
  • 能运行 Mini Agent 示例代码。
  • 能说明 OpenAI-compatible adapter 是为了解决什么问题。
  • 能给 Agent 加一个只读工具，并限制路径范围。
  • 能设计短期记忆、用户画像和长期情景记忆。
  • 能用 Markdown + 关键词检索做一个最小 RAG。

面试主线打卡

  • 能两分钟回答长期陪伴 AI 的记忆机制。
  • 能讲完整 RAG 数据工程链路。
  • 能解释长上下文为什么不会完全替代 RAG。
  • 能区分 FlashAttention、GQA、PagedAttention、RoPE Scaling 分别解决什么。

渲染地图主线打卡

  • 能区分 OpenLayers、Mapbox、Cesium 的适用场景。
  • 能解释 Map/View/Layer/Source。
  • 能解释 source/layer/style/vector tile。
  • 能解释 Entity、Primitive、DrawCommand。
  • 能说明自定义 shader 在 Cesium 中进入哪一层。

金融量化主线打卡

  • 能画出家庭、企业、政府、金融、城市和海外部门的社会运行总图。
  • 能解释政府支出、税收、央行基础货币、商业银行贷款之间的关系。
  • 能解释一线城市为什么工资高、房租高、服务价格高。
  • 能解释全球化中贸易、汇率、资本流动和美元体系如何影响国内经济。
  • 能解释稀缺性、机会成本、边际分析和激励。
  • 能画供需曲线，解释均衡价格、短缺、过剩和弹性。
  • 能说明现金流、时间价值、折现率和资产价格的关系。
  • 能解释 GDP、通胀、就业、利率、信用和资产价格如何互相传导。
  • 能解释 K 线不是预测器，而是周期成交结果压缩。
  • 能说明均线、MACD、RSI、KDJ 的底层输入是什么。
  • 能把一个买入信号拆成环境、位置、触发、确认、失效和仓位。
  • 能画出量化系统的数据、研究、回测、实盘和风控链路。
