# Learn Pi

Pi Agent 架构学习站，用来系统学习 `earendil-works/pi` 的核心架构、关键代码与阅读路线。

## 内容

- Pi Agent 四层架构地图
- `pi-ai`、`pi-agent-core`、`pi-coding-agent`、`pi-tui` 模块职责
- `agent-loop.ts`、`agent.ts`、`agent-session.ts`、`tools/index.ts` 等关键源码索引
- 可勾选学习路线与实战读源码任务

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 部署

仓库包含 GitHub Pages workflow。推送到 `main` 后，GitHub Actions 会把仓库根目录发布到 Pages。
